# Sistema de Gestión de Empresas ARL

Este script de Python es una herramienta de consola diseñada para gestionar un listado de empresas, almacenando sus datos en un archivo Excel. Permite añadir nuevas empresas, actualizar información existente y realizar cargas masivas de datos. Los campos de las empresas están estandarizados e incluyen validaciones para asegurar la integridad de la información, especialmente en relación con la clasificación de riesgos ARL.

## Características Principales

* **Persistencia en Excel:** Todos los datos se guardan y cargan desde un archivo Excel (`Listado_Empresas_ARL_Automatizado.xlsx`), manteniendo un formato tabular y legible.
* **Gestión de Datos:**
    * **Agregar Empresa:** Ingreso manual de datos de una empresa campo por campo con validaciones en tiempo real.
    * **Actualizar Empresa:** Búsqueda por "Razón Social" y edición interactiva de campos específicos.
    * **Carga Masiva:** Permite pegar múltiples líneas de datos de empresas desde la consola, siguiendo un formato pipe-separated (`|`), con validación de cada línea.
* **Validación de Campos:** Implementa reglas de validación para asegurar la calidad de los datos ingresados:
    * **Fechas:** Formato `DD/MM/AAAA`.
    * **Ingresos:** Solo números.
    * **Correos:** Formato de email estándar.
    * **Teléfonos:** Solo números, 7 o 10 dígitos.
    * **Tipo de Identificación:** Solo "NIT" o "NO NIT".
    * **Número de NIT:** Formato `XXXXXXXXX-X` o solo números. Si el "Tipo de Identificación" es "NO NIT", el campo "NUMERO_DE_NIT" se fuerza a "N/A".
    * **CIIU:** Número de 4 o 5 dígitos.
* **Manejo de Errores:** Proporciona mensajes claros para errores de validación o de formato en la entrada de datos.
* **Estilos y Formato de Excel:** Aplica automáticamente estilos a los encabezados del archivo Excel (negrita, fondo azul, centrado) y ajusta el ancho de las columnas para una mejor visualización.

## Requisitos

Para ejecutar este script, necesitarás tener Python instalado en tu sistema y las siguientes librerías:

* `openpyxl`: Para leer y escribir archivos Excel (.xlsx).
* `re`: Para expresiones regulares (usado en las validaciones de formato).
